# 老爸心脏康复计划 ❤️

> 用心守护，健康常伴

一个专为心脏手术后康复设计的个人健康管理应用，帮助术后患者科学康复、安全恢复。

## 📸 界面预览

应用包含爸爸的头像，温馨亲切的界面设计。

## ✨ 核心功能

### 1. 四阶段智能康复管理
- 🛡️ **保护期**（0-13天）：术后初期，轻微活动为主
- 💪 **恢复期**（14-55天）：逐步增加运动量
- ❤️ **强心期**（56-180天）：持续运动，增强心脏功能
- 🌟 **维护期**（180天+）：长期保持良好习惯

### 2. 每日任务打卡
- ✅ 戒烟监测（带死亡风险警告）
- ✅ 戒酒监测（带死亡风险警告）
- ✅ 按时吃药提醒
- ✅ 运动记录（自动调整目标）
- ✅ 血压心率监测（三级预警：正常/警告/危险）
- ✅ 症状记录（胸痛、胸闷、气短等）

### 3. 术后检查管理
- 🩺 自动生成检查计划（基于手术日期）
- 📊 LDL血脂监测（目标 < 1.4 mmol/L）
- ❤️ EF射血分数监测（目标 > 50%）
- ⏰ 检查倒计时提醒
- ✅ 达标情况智能判断

### 4. 强化管理模式
当检查指标未达标时（LDL ≥ 1.8 或 EF < 40），系统自动启动强化模式：
- 🏃 运动目标下调（更保守）
- 💊 用药提醒加强（红色高亮）
- 📈 需要更密切监测
- ⚠️ 阶段显示"强心期（强化版）"

### 5. 数据记录与分析
- 📈 连续无烟/无酒天数统计
- 📊 运动趋势分析
- 💓 血压心率历史记录
- 🚨 危险行为统计（近30天）

### 6. 视觉化进度
- 🗺️ 康复阶段路线图
- 📊 整体恢复进度百分比
- 🎯 当前阶段状态显示
- 🎨 彩色状态指示（绿/黄/红）

## 🛠️ 技术特点

- 📱 **纯前端应用**：单个HTML文件，无需服务器
- 💾 **本地存储**：使用浏览器LocalStorage，数据完全私密
- 🚀 **零成本部署**：可部署到GitHub Pages
- 📴 **离线可用**：无需网络连接即可使用
- 📱 **响应式设计**：适配手机、平板、电脑
- 🎨 **现代UI**：渐变色彩、动画效果、直观易用

## 🚀 使用方法

### 在线访问
访问部署后的GitHub Pages链接即可使用。

### 本地使用
1. 下载 `index.html` 和 `kaiqun-image.jpg` 到同一文件夹
2. 双击 `index.html` 用浏览器打开
3. 在"设置"页面填写手术日期
4. 开始每日打卡记录

## 📊 数据管理

### 数据存储
- 所有数据保存在浏览器LocalStorage中
- 三个数据键：
  - `heartRehabRecords`：每日记录
  - `heartRehabSettings`：设置信息
  - `heartRehabCheckups`：检查记录

### 数据备份
在"设置"页面可以：
- 📥 导出JSON数据文件
- 🗑️ 清空所有记录（需确认）

### 注意事项
⚠️ 数据仅存储在本地浏览器，清除浏览器数据会导致记录丢失，建议定期导出备份。

## 📝 术后检查时间表

系统会根据手术日期自动生成以下检查计划：

| 检查项目 | 推荐时间 | 监测指标 | 目标值 |
|---------|---------|---------|--------|
| 血脂检查1 | 术后4-8周 | LDL | < 1.4 mmol/L |
| 心电图+彩超 | 术后6-12周 | EF | > 50% |
| 血脂检查2 | 术后3个月 | LDL | < 1.4 mmol/L |
| 年度综合复查 | 术后12个月 | LDL + EF | 均达标 |
